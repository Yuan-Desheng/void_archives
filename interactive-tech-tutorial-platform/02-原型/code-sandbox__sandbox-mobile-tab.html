React 核心概念与 Hooks 第 2 章 · State 与生命周期

用 useState 让组件拥有"记忆"

普通 JS 变量在函数组件重新渲染时会被"重置"，useState 让 React 帮你在多次渲染之间"记住"这个值。

它返回一个数组：[值, 更新函数]，调用更新函数会触发组件重新渲染。

const [count, setCount] = useState(0);

切到「代码」Tab，把 Counter.js 里 useState(0) 的初始值改成 10，验证你理解了它的作用。
