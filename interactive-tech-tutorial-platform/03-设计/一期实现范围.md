## R8 三条 CI 门禁（一期必建，与后端无关）

| 门禁 | 手段 | 判据 |
|---|---|---|
| 零主题分支 | 静态扫描 `lib/engine/**`、`components/**` 的 `react\|vue\|vanilla` 字面量分支 | 命中=0 |
| 新增 JSON 零改动 | 放最小 `vue.json` 跑构建+渲染冒烟 | 构建过 + git diff 仅增 JSON |
| Schema 契约 | zod 校验全部 package JSON | 全过 |

> **R8 是成败判据**。一期没有后端，但 R8 一点不打折——它落在内容-引擎分离，与后端无关。
> 阶段④第一优先级 = 把 Package Loader + zod Schema + 这三条门禁建起来。

## 一期技术栈（收窄版）

Next.js 15 (App Router, RSC/SSG) + React 19 + TS 5 + Tailwind v4 + shadcn/ui + `@codesandbox/sandpack-react`。
**无** Drizzle / Postgres / Clerk / Route Handler。部署：Vercel 静态 + 环境变量（无 DB/Clerk 密钥）。

## 一期数据形态

- **内容**：`content/react.json`（TutorialPackage，随仓库）——唯一真源，纯静态。
- **进度**：`localStorage['ittp:progress']` = `{ [packageId]: { currentStepId, completedStepIds[], lastVisitedAt } }`。
- **无数据库、无用户表、无鉴权**。
