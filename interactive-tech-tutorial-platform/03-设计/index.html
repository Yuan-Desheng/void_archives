阶段③设计 · 说明书首页

互动式技术教程平台：内容与引擎分离、以 JSON 配置（TutorialPackage）驱动的「左讲解 + 右可运行 Sandpack 沙箱」内部自用自主学习工具。

对标 Vue3 官方互动教程的内部自用教程工具。核心价值是「内容与引擎分离」——教学内容以 JSON 配置驱动，引擎只做渲染与交互， 成败判据是「新增一个 vue.json 即可跑通新主题、引擎代码零改动」（R8）。首个主题 React（62 篇笔记/8 章，MVP 先精做 9 篇样板）， 预留 Vue/JS/算法扩展。产品形态经 2026-07-17 决策 D4 范式转向：忠于 Vue3「左讲解+右沙箱、靠内容引导、全程不拦人」，砍掉高亮/等待/聚光灯（R3/R4/R5）。 设计目标为内容引擎解耦、真实可运行沙箱、一键看答案（R10）、shadcn 灰阶克制视觉（D5）与三档响应式。
