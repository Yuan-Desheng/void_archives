代码高亮 · 双主题预览

同一段 React 代码在深色（One Dark 风格，沙箱默认）与浅色语法配色下的对照，代码块配色与页面亮暗主题解耦，保证可读性。

深色语法高亮（沙箱默认） 

function Counter() {
  const [count, setCount] = useState(0);
  // 每次点击 count 自增 1
  return (
    <button onClick={() => setCount(count + 1)}>
      点击次数：{count}
    </button>
  );
}
