继续学习提示与入口

嵌入式 UI 规范

本页非独立路由，定义「重新进入教程包」时基于本地进度展示续学横幅、以及课程库卡片完成度/按钮文案的呈现与交互规范。下方为可操作的实时演示区，切换场景可观察三种状态：学习中断点 / 已全部完成 / 首次访问。

演示控制

选择一个教程包并切换其本地进度场景，下方「续学横幅」与「课程库卡片」会实时联动更新（数据写入 localStorage · ittp:progress）。

续学横幅 / 轻提示

出现在「学习工作台」进入某教程包时的顶部区域

课程库教程包卡片

完成度百分比 + 按钮文案随进度切换（继续学习 / 开始学习 / 重新学习）

关键数据字段

字段 类型 说明
tutorialId string 教程包唯一标识，进度按此隔离存储，互不影响
currentStepId string 断点续学定位目标，点击「继续学习」跳转至此步骤
tutorialCompletionRate number (0-100) 教程包整体完成率，向下取整；驱动卡片百分比与按钮文案
lastVisitedAt timestamp 最后一次进入该教程包的时间，用于续学提示的相对时间展示
