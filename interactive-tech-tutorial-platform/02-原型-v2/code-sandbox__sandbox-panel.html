2.2 用 useState 管理组件状态

组件常常需要根据交互改变界面上显示的内容——比如点击一个按钮后，计数会增加。useState 是 React 提供的最基础的 Hook，它让函数组件拥有「记忆」：每次调用 setCount 都会告诉 React「这个组件的状态变了，请重新渲染」。 

const [count, setCount] = useState(0);

function handleIncrement() {
  setCount(count + 1);
}

右侧沙箱里已经放好了本节的练习：一个「点击 +1」的计数器组件，handleIncrement 函数还差最后一行——你需要自己补上让 count 加 1 的逻辑。

写完后点 Run 看看按钮是否真的能让数字增加。想不出来也没关系，点右侧的「给我看答案」直接看标准写法，看完还能一键切回你自己的版本继续想。

小结
  • useState 返回一个 [当前值, 更新函数] 数组
  • 调用更新函数会触发组件重新渲染
  • 不要直接修改 state 变量，必须通过更新函数
